What is a client?

A personal computer or a workstation that talks to the server. The communication consists of sending requests to and receiving responses from the server.

What is a server?

A special computer that hosts website files. Servers are like digital file cabinets. Unlike clients, servers have only one purpose: to host massive amounts of files and to serve those files to you.

What is an HTTP request?

A request sent to the server by the client, usually to retrieve, create, update, or delete data.

What is an HTTP response?

Information in the form of a text record, comprised of a status code, a header, and a body that a web server sends back to the client in response to receiving an HTTP request.

What is Django and why do we use it?

Django is an open-source web application framework, written in Python. A web framework, like Django, helps you build web apps. We use frameworks like Django to save ourselves from having to reinvent the wheel and to avoid some of the overhead when building websites.
